Avant de lancer, il faut bien vérifier que vous avez toutes les dépendances nécessaires : 

- **Pytorch**  
- **scikit-learn (sklearn)**  

Assurez-vous également que les répertoires **`train`** et **`test`** sont dans le même répertoire que les fichiers **`train+valid.py`** et **`prediction.py`**.

train+valid.py sert à entrainer et enregistrer le model

prediction.py sert à predire les images du répertoire 'test' et creer fichier .json
